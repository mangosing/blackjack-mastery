# 1. MVP Scope

Defines everything included in the first public release.

Included

- Basic Strategy Trainer
- Interactive Practice
- Explanations
- Progress Tracking
- Weak Hand Detection
- Dashboard

Excluded

- Card Counting
- Multiplayer
- AI Coach
- Mobile Apps

# 2. Functional Requirements

Each functional requirement should receive a unique identifier.

Example:

FR-001

The system shall display a random blackjack hand.

FR-002

The system shall allow the user to choose Hit, Stand, Double, Split or Surrender.

FR-003

The system shall evaluate the selected action.
