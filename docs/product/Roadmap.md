# Purpose

This document describes the planned evolution of Blackjack Mastery from its initial MVP to a comprehensive blackjack learning platform.

The roadmap provides strategic direction rather than fixed delivery dates. Priorities may change as we gather user feedback.

---

# Product Vision

Become the world's leading educational platform for blackjack.

---

# Guiding Principles

Every roadmap decision should support one or more of the following:

- Better learning
- Better retention
- Better user experience
- Measurable progress
- Long-term mastery

# Phase 1 — MVP

## Goal

Teach perfect basic strategy.

### Features

- Basic Strategy Academy
- Interactive Practice
- Immediate Feedback
- Explanations
- Dashboard
- Progress Tracking
- Weak Skill Detection

Success Criteria

A new player can learn and practice basic strategy from beginning to end.

---

# Phase 2 — Adaptive Learning

## Goal

Personalize learning.

### Features

- Learning Engine
- Skill Recommendations
- Weak Skill Review
- Mastery Tracking
- Daily Practice
- Streaks

Success Criteria

Every user receives personalized practice sessions.

---

# Phase 3 — Card Counting Academy

## Goal

Teach Hi-Lo card counting.

### Features

- Running Count
- True Count
- Deck Estimation
- Speed Drills
- Count Accuracy
- Betting Ramp Introduction

Success Criteria

Users can accurately maintain a running count and convert to a true count under realistic conditions.
